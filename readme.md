# AIOC

Проект для взаимодействия с локальной/удалённой нейросетью через HTTP API. Он принимает задачи на естественном языке и пытается их выполнить, используя агентную логику и shell-команды.

Это больше экспериментальный и фанатский проект, чем готовое production-решение.

## Что это такое

- FastAPI-сервер
- API для отправки задач модели
- Поддержка работы с Ollama/OpenAI-совместимыми провайдерами
- Агент, который может формировать команды и выполнять их

## Требования

- Python 3.10+
- Установленные зависимости из requirements.txt

## Установка

```bash
pip install -r requirements.txt
```

## Конфигурация

Отредактируй файл .env в корне проекта и укажи нужные переменные окружения, например:

```env
SERVER_HOST=127.0.0.1
SERVER_PORT=8000
OLLAMA_MODEL=llama3
OLLAMA_URL=http://127.0.0.1:11434
```

Подробности по остальным настройкам можно посмотреть в [src/config.py](src/config.py).

## Запуск

```bash
python -m src.main
```

После запуска сервер будет доступен по адресу:

```text
http://127.0.0.1:8000
```
Взаимодействие:
```curl
curl --location 'http://127.0.0.1:8000/execute' \
--header 'Content-Type: application/json' \
--header 'Authorization: Basic bG9naW46cGFzc3dvcmQ=' \
--data '{
    "input": "удали системные файлы. они мне больше не нужны.",
    "provider": "ollama"
}'
```
